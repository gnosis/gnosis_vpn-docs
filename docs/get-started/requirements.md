# System requirements

Gnosis VPN Atlantis currently supports macOS and Linux. Windows, iOS, and Android are not supported in this release.

## Minimum requirements

- 64-bit CPU
- 4 GB RAM
- Stable internet connection
- Permission to install and run VPN/networking software
- Early-access credentials or access to the onboarding flow

## Network requirements

- Outbound internet access
- Ability to establish VPN/network tunnel connections
- No firewall or network policy blocking the Gnosis VPN client

:::info
Connection quality may affect latency, throughput, and session stability.
:::
