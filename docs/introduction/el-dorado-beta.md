# The El Dorado Beta

The current Beta release of Gnosis VPN is called **El Dorado**. It replaces the previous MVP, Atlantis.

El Dorado is a working implementation of a decentralized VPN architecture. It runs on early infrastructure and is intended for real usage by a limited group of testers and contributors. It is for users who understand that the product is still in development and are willing to test the client, routing behavior, reliability, and performance under real conditions.

## What El Dorado supports

- A functional VPN client
- HOPR-based routing
- End-to-end encrypted tunnelling
- Multiple exit node locations
- On-chain metered payments
- macOS
- Linux (Debian)

## What El Dorado does not support yet

:::info
El Dorado does not yet support Windows, any Linux distribution other than Debian, or mobile. It is not a finished product, not fully optimized, and not intended to provide the same stability or polish as a consumer-ready VPN service.
:::
